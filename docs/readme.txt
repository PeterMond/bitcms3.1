|     bitcms 产品使用说明               |
 ----------------------------------------

一、平台需求

IIS/.NET Framework 4.5 + MySQL5


基本目录结构
/
../App_Data 日志文件目录
../Areas 主要后台管理文件目录
../bin     类库文件目录
../cache 文件缓存目录[必须有可写入权限]
../config 配置文件目录[必须有可写入权限]
../css 样式文件目录
../images 图片目录 
../install     安装程序目录，安装完后可删除[安装时必须有可写入权限]
../js js脚本目录和bootstrap目录
../ueditor 百度编辑器目录
../upload 附件上传目录[必须有可写入权限]
../Views 前台视图目录
../Global.asax Global文件
../web.config 配置文件[安装时必须有可写入权限]



二、程序安装使用
1.下载程序解压到本地目录;
2.上传程序目录中的/bitcms.web3.1到网站根目录
3.运行http://www.yourname.com/admin/install/(yourname表示你的域名),按照安装提速说明进行程序安装
